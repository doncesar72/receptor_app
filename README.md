# РЕЦЕПТОР

Умное приложение для создания рецептов на основе фотографий холодильника.

## Описание

РЕЦЕПТОР — это инновационное веб-приложение, которое использует искусственный интеллект для анализа содержимого вашего холодильника и генерации персонализированных рецептов. Просто сфотографируйте продукты, и AI предложит 5 уникальных рецептов, которые можно приготовить из имеющихся ингредиентов.

## Возможности

- 📸 **Анализ фотографий** - AI распознает продукты в вашем холодильнике
- 🤖 **Умные рецепты** - Генерация 5 уникальных рецептов на основе найденных продуктов
- ⏱️ **Разная сложность** - От быстрых закусок до полноценных ужинов
- ❤️ **Избранное** - Сохраняйте любимые рецепты
- 📱 **Адаптивный дизайн** - Работает на всех устройствах
- 🎨 **Красивый интерфейс** - Современный дизайн с градиентами и анимациями

## Технологии

- **Frontend**: Next.js 16, TypeScript, Tailwind CSS
- **AI**: OpenAI GPT-4o для анализа изображений и генерации рецептов
- **UI**: shadcn/ui компоненты
- **Backend**: Next.js API routes

## Установка

```bash
# Клонировать репозиторий
git clone <repository-url>
cd fridge_chef

# Установить зависимости
npm install

# Создать .env.local файл с переменными окружения
cp .env.example .env.local

# Запустить development сервер
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Переменные окружения

Создайте файл `.env.local` со следующими переменными:

```env
OPENAI_API_KEY=your_openai_api_key
OPENAI_MODEL=gpt-4o
OPENAI_TEMPERATURE=0.8
OPENAI_MAX_TOKENS=3000
ADMIN_KEY=receptor_admin_2024
```

## Функции

### 📸 Анализ холодильника
- Загрузка фотографии холодильника
- AI распознавание продуктов
- Генерация списка найденных ингредиентов

### 🍳 Генерация рецептов
- 5 уникальных рецептов на основе найденных продуктов
- Разные категории: быстрые, полноценные, креативные
- Детальная информация: время, калории, сложность

### 📱 Детальная страница рецепта
- Пошаговые инструкции
- Интерактивный список ингредиентов
- Кнопка "Начать готовить" с плавной прокруткой

### ❤️ Избранное
- Сохранение любимых рецептов
- Быстрый доступ из главного меню
